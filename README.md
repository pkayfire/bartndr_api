## Getting Started

Install the module with: `npm install bartndr-api`

```js
var bartndr-api = require('bartndr-api');
bartndr-api.awesome(); // "awesome"
```

Install with cli command

```sh
$ npm install -g bartndr-api
$ bartndr-api --help
$ bartndr-api --version
```


```sh
# creates a browser.js
$ grunt browserify
```
